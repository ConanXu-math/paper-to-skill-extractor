# Examples

These examples show generated outputs from `paper-to-skill-extractor`.

Each case includes:

```text
report.md
skill_candidates.json
skill_cards/*.yaml
```

The original `paper.md` files are intentionally omitted. In real runs, the skill preserves the full input paper at `outputs/<paper_id>/paper.md`, and every candidate references line numbers in that file.

## Included Cases

- `2510.08929v1`: proof-pattern extraction from a paper on mirror flow matching with heavy-tailed priors.
- `2602.05174v1`: proof-pattern extraction from a paper on total variation rates for Riemannian flow matching.

Use these outputs as format examples, not as replacements for running the skill on your own `paper.md`.
